# 日本観光統計

このリポジトリは、都道府県および市区町村別に集計された、日本の観光来訪者数の自動更新される統計データを提供します。データは単一の使いやすいCSVファイルに統合されています。

## データソース

データは、観光庁が提供する[デジタル観光統計オープンデータ](https://www.nihon-kankou.or.jp/home/jigyou/research/d-toukei/)から取得しています。

## 特徴

- **月次データ**: 日本の各都道府県および主要都市の観光来訪者数。
- **自動更新**: GitHub Actionsのワークフローが毎日実行されて最新のデータをダウンロードし、データセットを最新の状態に保ちます。
- **統合CSV**: 個別のデータファイルはすべて処理され、分析しやすいように単一のファイル `data/all.csv` に統合されます。
- **Denoによる処理**: データ処理パイプラインはDenoで構築されています。

## データ形式 (`data/all.csv`)

主な出力ファイルは `data/all.csv` で、以下の列を含みます。

| 列名 | 説明 | 例 |
| :--- | :--- | :--- |
| `month` | データの年月（`YYYYMM`形式）。 | `202201` |
| `lgcode` | 地域の地方公共団体コード。 | `1101` |
| `n` | その月の総来訪者数。 | `230903` |

## 自動化

このリポジトリは、`.github/workflows/update.yml` で定義されたGitHub Actionsワークフローを使用しています。このワークフローは毎日スケジュール実行され、以下の処理を行います。

1. データソースのページをスクレイピングして新しいCSVファイルを探します。
2. 新しいファイルをダウンロードします。
3. 統合ファイルである `data/all.csv` を再構築します。
4. 更新されたデータをコミットし、リポジトリにプッシュします。

## ローカルでの利用

データ更新処理はローカルでも実行できます。

### 必要条件

- [Deno](https://deno.land/)

### 手順

1. **リポジトリのクローン:**
    ```bash
    git clone https://github.com/your-username/japan-kanko-stat.git
    cd japan-kanko-stat
    ```

2. **Denoスクリプトの実行:**
    ```bash
    # 最新のソースCSVファイルを /data ディレクトリにダウンロードします
    deno run -A download.js

    # ダウンロードしたファイルを data/all.csv に統合します
    deno run -A make.js
    ```
